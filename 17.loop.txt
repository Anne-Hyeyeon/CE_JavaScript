21강. 반복문
[목적] 반복되는 코드를 반복문 while을 사용하여 간단히 작성한다.

[내용]
- while(소괄호 안에는 블리언값이 들어간다. true or false값이 나오게 지정해준다.){
  중괄호 안에는 true값이 나오면 반복 출력할 내용을 적는다.
}
false값일 때부터 출력될 내용

-> 블리온 데이터 값이 트루이면 중괄호 {}안 코드가 false가 될 때까지 반복 출력되고, false가 되었을 때 이후 코드가 출력된다.

while은 실행 순서를 제어하는 제어문이다. (if같은 존재)
소괄호 안에 반복문이 언제 종료될지를 잘 지정하는게 중요함.

- 코드가 몇 번 실행됐는지 적어놔야 하는데 그걸 하기에 좋은 것이 변수다.
관습적으로 i라는 변수에 반복문의 조건을 주는 변수를 할당해준다.

var i = 0; -> i 변수는 0이다.


[코드]

(1)
<script>
      document.write('<li>1</li>');
      var i = 0; 

-> i 라는 변수에 0을 지정

(2)
      while(i < 3) {
        document.write('<li>2</li>');
        document.write('<li>3</li>');
        i = i + 1;
      }

-> i라는 변수가 3보다 작으면 {}안의 코드를 반복 출력.
i = i + 1; -> (기존 i의 값에 1을 더한 결과를 i의 새로운 값으로 준다.)
이 코드에 온 순간에 i값에 1을 더하면 변수 i값이 바뀐다.
0, 1, 2 까지 3번 반복문을 출력하다가 i가 3이 되는 순간 반복문이 끝난다.

      document.write('<li>4</li>');
</script>

[참조]
while (i<3) 은 실행이 되나 while(i&lt3)은 실행이 안되는 이유
&lt는 html의 문법에서 쓰는 것이기에 js에서는 <,>로 처리가 가능하다.

데이터타입 - 문자열과 숫자 (String and Number)

- Javascript data type으로 구글 검색하면 데이터타입에 대해 알 수 있다.

Number
- 말 그대로 숫자 타입의 데이터
- 웹브라우저 Inspect - Console에서 alert(1);을 쳐보자. 경고창으로 1이 뜬다.
- 그렇다면 alert(1+1)은? 2가 경고창으로 뜬다.
- 경고창이 계속 나오는 게 번거롭다? 그럼 그냥 콘솔에다가 1+1을 치자. 2가 결과값으로 뜬다. 
- +는 산술연산자에 속한다. (그 외에도 -. *, / 등이 있다.)

String
- 따옴표로 시작해서 따옴표로 끝나는 데이터 (큰따옴표는 큰따옴표로, 작은따옴표는 작은따옴표로 끝나면 된다.)
- 문자 데이터를 이용한 기능은 어마어마하게 많다.
- ex. 문자 데이터 뒤에 .length 붙이면 문자 개수를 알려준다.
- MDN에서 문자 데이터를 활용한 기능을 찾을 수 있다. 
- 이때 length와 같은 걸 'properties'라고 부른다.
- Method 예시
1) toUpperCase() 소문자->대문자 / 사용하는 방법은 '본문' .toUpperCase() 입력
2) indexOF() .문자열(strng)에서 특정 문자열(searchvalue)을 찾고, 검색된 문자열이 '첫번째'로 나타나는 위치 index를 리턴함.


- 1은 숫자 데이터지만, "1"은 문자 데이터.
- 1+1은 2로 출력되고, "1"+"1"은 11로 출력됨.
